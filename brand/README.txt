Trail - Asset Package
=====================

Files
-----
icon-color.svg            Primary icon, 512x512, rounded-square ocean-gradient tile. Use for app/org avatars.
icon-monochrome-black.svg Flat single-color mark (ink), transparent background. For light surfaces / print.
icon-monochrome-white.svg Flat single-color mark (white), transparent background. For dark surfaces.
wordmark.svg              "trail" wordmark, ink, transparent background. For light surfaces.
wordmark-reversed.svg      "trail" wordmark, ocean-accent color. For dark surfaces.
lockup-horizontal-light.svg  Icon + wordmark side by side, for light surfaces.
lockup-horizontal-dark.svg   Icon + wordmark side by side, on a dark ocean panel.
favicon.svg                Tightly-cropped vector favicon; legible from 512px down to 16px.

Colors

Typeface
--------

JetBrains Mono (labels, code-adjacent UI) - https://fonts.google.com/specimen/JetBrains+Mono

Usage
-----
Keep clearspace around the mark at least equal to the waypoint node's radius.
Never stretch, rotate, recolor arbitrarily, or add shadows/bevels to the mark.
The icon is safe within a circular crop (GitHub org avatar) as shipped.
